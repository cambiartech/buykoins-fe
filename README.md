# Buy TikTok Coins - Creator Earnings Platform

A platform for TikTok creators to withdraw their earnings through a secure agency system.

## Features

- **Landing Page**: Modern, TikTok-themed landing page with gradient backgrounds
- **User Dashboard**: Wallet management, bank account linking, TikTok account connection, and withdrawal requests
- **Admin Dashboard**: Withdrawal request management, verification, and PayPal onboarding

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React Icons** - Icon library

## Design

- **Primary Color**: #e81155 (TikTok Pink)
- **Typography**: ABC Monument Grotesk & Sequel Sans
- **Theme**: TikTok-inspired with gradients, white, and black
- **Mobile-First**: Responsive design for all devices

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Pages

- `/` - Landing page
- `/signup` - User signup (Email/Google/TikTok)
- `/login` - User login
- `/dashboard` - User dashboard
- `/admin` - Admin dashboard

## Notes

- Font files for ABC Monument Grotesk and Sequel Sans should be added to `/public/fonts/` directory
- Backend API integration is pending - frontend is ready for API connections
- All forms and actions are currently mock implementations
