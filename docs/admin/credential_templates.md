## How to Use These Templates

1. **Open Support Chat**: Navigate to `/admin/support` in the admin dashboard
2. **Select User Conversation**: Find and select the user's conversation (or create a new one)
3. **Copy Template**: Copy the appropriate template from this document
4. **Fill in Details**: Replace placeholders like `[User Name]`, `[username]`, `[password]`, etc. with actual values
5. **Send Message**: Paste the filled template into the message input and send
6. **Track Status**: Monitor the conversation to see when the user confirms receipt

## Security Best Practices

‚ö†Ô∏è **IMPORTANT**: When sending credentials:

1. **Verify User Identity**: Always confirm you're messaging the correct user
2. **Use Secure Channels**: Only send credentials through the support chat system (not email or external messaging)
3. **Encourage Password Change**: Always remind users to change passwords after setup
4. **Document in Notes**: Record credential details in the onboarding completion notes for audit purposes
5. **Delete After Use**: Consider deleting credential messages after user confirms receipt (if supported)

## Notes for Onboarding Completion

When completing onboarding, include credential details in the notes field:

**Format:**
```
TikTok Account: Username: [username], Password: [password]
Payment method configured on [date]
User confirmed receipt of credentials on [date]
```

**Example:**
```
TikTok Account: Username: topsky, Password: lantern
Payment method configured on 2025-01-23
User confirmed receipt of credentials on 2025-01-23
```
